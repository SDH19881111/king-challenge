host.js의 명단 불러오기 로직은 다음 JSON 형식을 지원한다.

형식 1:

[
  "민준",
  "서연",
  "지후"
]

형식 2:

{
  "students": [
    { "name": "민준" },
    { "name": "서연" },
    { "name": "지후" }
  ]
}

형식 3:

{
  "students": [
    { "이름": "민준" },
    { "이름": "서연" },
    { "이름": "지후" }
  ]
}

Apps Script에서 CORS 문제가 생기면 doGet 함수가 JSON을 ContentService로 반환하도록 한다.

예시:

function doGet() {
  const sheet = SpreadsheetApp.getActive().getSheetByName("명단");
  const values = sheet.getDataRange().getValues();
  const headers = values.shift();
  const nameIndex = headers.indexOf("이름");
  const students = values
    .map(row => ({ name: row[nameIndex] }))
    .filter(student => student.name);

  return ContentService
    .createTextOutput(JSON.stringify({ students }))
    .setMimeType(ContentService.MimeType.JSON);
}

추가하면 좋은 것:

- 번호 컬럼 지원
- 학급 컬럼 지원
- 결석 컬럼을 구글시트에서 읽어 초기 absent 값으로 반영

6. 학생용 일반 대결 점수 로직

일반 대결:

- 승리 학생: +2점
- 패배 학생: -1점
- 패배 학생 점수는 0점 아래로 내려가지 않음

예:

winner.score = winner.score + 2
loser.score = Math.max(0, loser.score - 1)

만점 도달 처리:

- 만점 = 게임 종목 수 * 2점
- 학생 점수가 만점 이상이 되면 reachedPerfectAt 기록
- 왕 자리 수가 남아 있으면 status = king
- 왕 자리가 꽉 찼으면 status = challenger
- 도전자 대기열은 reachedPerfectAt 순서대로 정렬

주의:

- 결석생은 학생용 선택 목록에서 제외
- 결석생은 왕관 계산에서도 제외


7. 왕 vs 도전자 결전 로직

학생용 화면에서 도전자 결전 모드를 지원해야 한다.

조건:

- 도전자 status가 challenger인 학생이 있어야 한다.
- 왕 status가 king인 학생이 있어야 한다.

흐름:

1. 도전자가 자신 있는 게임 종목 선택
2. 도전자가 대결할 왕 선택
3. 결과 입력
4. 이긴 사람을 누르면 점수와 왕관 상태 변경

점수 로직:

- 도전자가 패배할 경우:
  도전자 점수 = 총 게임 종목 수
  예: 종목 5개면 5점

- 왕이 패배할 경우:
  왕 점수 = Math.round(총 게임 종목 수 / 2)
  예: 종목 5개면 Math.round(2.5) = 3점
  왕은 왕관을 잃음
  이긴 도전자는 새 왕이 됨

구현 팁:

- 일반 대결과 결전 대결을 match.type으로 구분한다.
- 일반 대결: type = "normal"
- 결전 대결: type = "challenge"

왕이 패배했을 때는 단순 점수 계산 후 recalculateRanks만 호출하면 도전자가 더 늦게 만점에 도달했기 때문에 기존 왕 순서가 꼬일 수 있다.
따라서 결전 승리 도전자를 즉시 왕으로 승격시키는 별도 함수가 필요하다.

추천 함수:

- resolveNormalMatch(match, winnerId)
- resolveChallengeMatch(match, winnerId)
- promoteChallengerToKing(challengerId, defeatedKingId)
- recalculateRanks()

9. 사운드 작업

현재 host.js는 Web Audio API로 간단한 효과음을 낸다.

나중에 실제 파일을 쓰려면 아래 폴더를 만든다.

outputs/assets/sounds

추천 파일:

- click.mp3
- win.mp3
- lose.mp3
- crown.mp3
- reset.mp3

예시 코드:

const sounds = {
  click: new Audio("./assets/sounds/click.mp3"),
  win: new Audio("./assets/sounds/win.mp3"),
  lose: new Audio("./assets/sounds/lose.mp3"),
  crown: new Audio("./assets/sounds/crown.mp3"),
  reset: new Audio("./assets/sounds/reset.mp3")
};

function playSound(name) {
  const sound = sounds[name];
  if (!sound) return;
  sound.currentTime = 0;
  sound.play();
}
